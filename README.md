docsang
=======

使用trans来管理markdown生成和预览

## install

前端依赖bower

```
npm install -g bower
cd frontend
npm install
bower install
```

安装服务器依赖

```
cd ../server
npm install
```

## start server

```
cd server 
npm start
```


## start web

```
cd frontend 
npm start
```

open http://127.0.0.1:8000/
